# habit_monitor

How to run?
1. Clone the repository.
2. Run "npm init"
3. Run "npm install"
4. Run "npm start"
5. In Browser, Run http://localhost:8000


This Habit monitor app is developed using Nodejs, expressjs and mongoDB.

--> You can add habits
--> Mark completed or uncompleted.
--> Weekly tracking
--> Remove habits
